C: Content Hub

Closest to simonwillison.net. Search + tag cloud + content stream. No sidebar, no persistent chat. Chat is a separate page accessed via "ASK AGENT" button. Simplest, most content-focused. Tag-based discovery like Simon's site.

Typography Recommendation

Inter for body text (readable at density, variable weight) + JetBrains Mono for labels, metadata, and code. Simon Willison uses system fonts — Inter gives you that same clean, information-dense readability while keeping JetBrains Mono for the technical/tactical feel in metadata.
